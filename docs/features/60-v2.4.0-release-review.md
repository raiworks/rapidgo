## Decisions Made

| Decision | Rationale |
|----------|-----------|
| Keep `config.Load()` in serve's `RunE` | Must read `RAPIDGO_MODE` before `NewApp()` — idempotent duplicate is harmless |

| Copy `GlobalHandlers()` loop over `Use(globalHandlers...)` | `Use()` wraps in middleware chain; direct copy preserves original handler order |
