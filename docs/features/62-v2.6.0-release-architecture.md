## 1. ENH-06: Logger Interface

### Current State

```go
// core/logger/logger.go
func Setup() *slog.Logger {
    // ... builds handler, creates *slog.Logger, calls slog.SetDefault() ...
    return logger
}
```

No interface. Framework internals call `slog.Info()`, `slog.Error()` globally (14 call sites across `core/cli/`, `core/server/`, `core/middleware/`). Users cannot swap logging backend.

### Design

Add a `Logger` interface and a default `SlogLogger` implementation. **Do NOT touch the 14 global `slog.*` call sites** — those continue to work via `slog.SetDefault()`. The interface is for callers who want to inject/mock a logger.

```go
// Logger defines the logging contract for RapidGo.
// The default implementation wraps slog. Users can implement this
// interface to plug in Zap, Zerolog, or a test spy.
type Logger interface {
    Debug(msg string, args ...any)
    Info(msg string, args ...any)
    Warn(msg string, args ...any)
    Error(msg string, args ...any)
    With(args ...any) Logger
}
```

**Default slog adapter:**

```go
// SlogLogger wraps *slog.Logger to implement the Logger interface.
type SlogLogger struct {
    log *slog.Logger
}

// NewSlogLogger creates a Logger wrapping the given *slog.Logger.
func NewSlogLogger(l *slog.Logger) *SlogLogger {
    return &SlogLogger{log: l}
}

func (s *SlogLogger) Debug(msg string, args ...any) { s.log.Debug(msg, args...) }
func (s *SlogLogger) Info(msg string, args ...any)  { s.log.Info(msg, args...) }
func (s *SlogLogger) Warn(msg string, args ...any)  { s.log.Warn(msg, args...) }
func (s *SlogLogger) Error(msg string, args ...any) { s.log.Error(msg, args...) }

func (s *SlogLogger) With(args ...any) Logger {
    return &SlogLogger{log: s.log.With(args...)}
}
```

**Updated Setup():**

```go
// Setup initializes the global slog logger and returns a Logger interface.
// Reads LOG_LEVEL, LOG_FORMAT, LOG_OUTPUT from environment.
// Sets slog.SetDefault() so global slog.Info() etc. still work.
func Setup() Logger {
    // ... existing handler/writer logic unchanged ...
    l := slog.New(handler)
    slog.SetDefault(l)
    return NewSlogLogger(l)
}
```

**Return type change**: `*slog.Logger` → `Logger`. This is **NOT breaking** because:
- The `Logger` interface matches slog's method signatures (`Debug/Info/Warn/Error` with `(string, ...any)`)
- Callers that assign to `*slog.Logger` will break at compile time, but the starter app's `LoggerProvider.Boot()` discards the return value (`logger.Setup()` — no assignment)
- Any caller needing the raw `*slog.Logger` can use `slog.Default()` after setup

### Migration for callers assigning the return value

If a caller does `l := logger.Setup()` and expects `*slog.Logger`, they can either:
1. Use the `Logger` interface (recommended)
2. Call `slog.Default()` to get the underlying `*slog.Logger`

### What stays the same

- `Close()` function — unchanged
- `parseLevel()` function — unchanged 
- `level.go` file — unchanged
- All 14 global `slog.*` call sites in framework — unchanged (work via `slog.SetDefault`)
- log file setup code — unchanged

## 2. ENH-05: Lightweight Typed Config `LoadConfig[T]()`

### Current State

```go
// core/config/config.go
func Load() { godotenv.Load() }

// core/config/env.go
func Env(key, fallback string) string
func EnvInt(key string, fallback int) int
func EnvBool(key string, fallback bool) bool
```

No struct-based config. Users manually build config structs field by field.

### Design

Add a generic `LoadConfig[T]()` function that reads env vars into a struct using struct tags:

```go
// LoadConfig reads environment variables into a struct T using struct tags.
//
// Supported tags:
//   - `env:"KEY"` — env var name to read (required)
//   - `default:"value"` — fallback if env var is empty/unset
//   - `validate:"required"` — passed to go-playground/validator
//
// Supported field types: string, int, bool, float64, time.Duration, []string (comma-separated).
//
// Example:
//
//	type AppConfig struct {
//	    Port    int           `env:"PORT" default:"8080" validate:"min=1,max=65535"`
//	    Debug   bool          `env:"APP_DEBUG" default:"false"`
//	    Name    string        `env:"APP_NAME" default:"app"`
//	    Timeout time.Duration `env:"TIMEOUT" default:"30s"`
//	}
//	cfg, err := config.LoadConfig[AppConfig]()
func LoadConfig[T any]() (T, error) {
    var cfg T
    v := reflect.ValueOf(&cfg).Elem()
    t := v.Type()

    for i := 0; i < t.NumField(); i++ {
        field := t.Field(i)
        fv := v.Field(i)

        envKey := field.Tag.Get("env")
        if envKey == "" {
            continue
        }

        raw := os.Getenv(envKey)
        if raw == "" {
            raw = field.Tag.Get("default")
        }
        if raw == "" {
            continue
        }

        if err := setField(fv, raw); err != nil {
            var zero T
            return zero, fmt.Errorf("config: field %s: %w", field.Name, err)
        }
    }

    validate := validator.New()
    if err := validate.Struct(cfg); err != nil {
        var zero T
        return zero, fmt.Errorf("config: validation failed: %w", err)
    }

    return cfg, nil
}
```

**Type setter helper** (unexported):

```go
func setField(fv reflect.Value, raw string) error {
    switch fv.Kind() {
    case reflect.String:
        fv.SetString(raw)
    case reflect.Int, reflect.Int64:
        if fv.Type() == reflect.TypeOf(time.Duration(0)) {
            d, err := time.ParseDuration(raw)
            if err != nil {
                return err
            }
            fv.Set(reflect.ValueOf(d))
        } else {
            n, err := strconv.ParseInt(raw, 10, 64)
            if err != nil {
                return err
            }
            fv.SetInt(n)
        }
    case reflect.Bool:
        fv.SetBool(raw == "true" || raw == "1")
    case reflect.Float64:
        f, err := strconv.ParseFloat(raw, 64)
        if err != nil {
            return err
        }
        fv.SetFloat(f)
    case reflect.Slice:
        if fv.Type().Elem().Kind() == reflect.String {
            parts := strings.Split(raw, ",")
            for i := range parts {
                parts[i] = strings.TrimSpace(parts[i])
            }
            fv.Set(reflect.ValueOf(parts))
        }
    default:
        return fmt.Errorf("unsupported type %s", fv.Kind())
    }
    return nil
}
```

## 3. ENH-02a: Additive `GenerateTokenFromString()`

### Current State

```go
// core/auth/jwt.go
func GenerateToken(userID uint) (string, error) {
    // ... reads JWT_SECRET, JWT_EXPIRY ...
    claims := jwt.MapClaims{
        "user_id": userID,  // uint
        "exp":     ...,
        "iat":     ...,
    }
    // ...
}
```

### Design

Add `GenerateTokenFromString()` alongside the existing function. Identical logic, different parameter type:

```go
// GenerateTokenFromString creates a signed JWT for the given user ID string.
// Use this for UUID or other string-based primary keys.
// Reads JWT_SECRET and JWT_EXPIRY (seconds) from environment.
func GenerateTokenFromString(userID string) (string, error) {
    secret := os.Getenv("JWT_SECRET")
    if secret == "" {
        return "", errors.New("JWT_SECRET is not set")
    }
    if len(secret) < 32 {
        return "", errors.New("JWT_SECRET must be at least 32 bytes")
    }

    expiry := 3600
    if v := os.Getenv("JWT_EXPIRY"); v != "" {
        if parsed, err := strconv.Atoi(v); err == nil {
            expiry = parsed
        }
    }

    claims := jwt.MapClaims{
        "user_id": userID,  // string — for UUID support
        "exp":     time.Now().Add(time.Duration(expiry) * time.Second).Unix(),
        "iat":     time.Now().Unix(),
    }

    token := jwt.NewWithClaims(jwt.SigningMethodHS256, claims)
    return token.SignedString([]byte(secret))
}
```

**Internal refactoring**: Extract shared JWT logic into an unexported helper to avoid code duplication:

```go
// jwtConfig reads JWT_SECRET and JWT_EXPIRY from environment.
func jwtConfig() (secret string, expiry int, err error) {
    secret = os.Getenv("JWT_SECRET")
    if secret == "" {
        return "", 0, errors.New("JWT_SECRET is not set")
    }
    if len(secret) < 32 {
        return "", 0, errors.New("JWT_SECRET must be at least 32 bytes")
    }

    expiry = 3600
    if v := os.Getenv("JWT_EXPIRY"); v != "" {
        if parsed, err := strconv.Atoi(v); err == nil {
            expiry = parsed
        }
    }
    return secret, expiry, nil
}
```

Then both functions call it:

