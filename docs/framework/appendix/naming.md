# Framework Naming

## Abstract

Candidate names considered for the framework, along with naming
criteria.

## Candidates

| Name | Notes |
|------|-------|
| Grove | Nature-themed, short |
| Gonix | Go + Unix connotation |
| GoNest | Go + Nest (structure) |
| FluxGo | Flow/change + Go |
| OrbitGo | Orbit + Go |
| NovaGo | Nova (new) + Go |
| GoCraft | Craftsmanship + Go |

## Naming Considerations

- **Uniqueness** — Must not conflict with existing Go packages or
  frameworks
- **Discoverability** — Easy to search for; distinctive
- **Brevity** — Short, memorable, easy to type
- **Domain availability** — Check `.dev`, `.io`, `.com`
- **Go convention** — Lowercase, no hyphens in package names

## Selected Name

**RapidGo** — selected by RAiWorks.
